# Fitness Rival

A gamified fitness application designed to make working out more engaging and social. Currently in active development.

## Project Overview

Fitness Rival aims to transform the fitness experience by combining gamification elements with social features, making exercise more motivating and enjoyable. The app will allow users to compete with friends, earn achievements, and track progress in a fun, engaging way.

## The Build (Seth)

- Designing user experience and gamification mechanics
- Setting up development environment
- Building core features and user interface

## The Break (Christine)

- Developing testing strategy for gamification features
- Planning performance testing for real-time social features
- Designing user experience testing protocols
- Establishing quality metrics for fitness tracking accuracy

## Planned Features

- **Gamification**: Points, badges, and achievements system
- **Social Features**: Compete with friends, share achievements
- **Workout Tracking**: Log exercises, sets, and reps
- **Progress Visualization**: Charts and graphs showing fitness journey
- **Challenges**: Weekly and monthly fitness challenges
- **Leaderboards**: See how you rank among friends and community
